Features

Powerful tracking

Auto-Watch

Point TENET at any directory and it instantly begins tracking changes. No git init, no commits, no friction.

Version Timeline

Every change creates an immutable snapshot. Browse the full history of any file on a beautiful visual timeline.

Instant Restore

Restore any file to any previous version with a single click. No complicated merge conflicts or checkout steps.

Diff Viewer

Inline diff comparison lets you see exactly what changed between versions - additions, deletions, line by line.

Content-Addressed

SHA-256 hashed blobs with automatic deduplication. Identical content is stored only once, saving disk space.

File Explorer

Browse all tracked files in a tree view. Filter and search instantly. Click any file to jump to its history.
